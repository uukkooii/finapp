# 个人财务规划管理 App — 设计方案

## 基本信息
- **App 名称**：金库（暂定）
- **平台**：Android + iOS
- **技术栈**：Flutter + SQLite（本地存储，隐私安全）
- **设计风格**：简洁专业，深色主题，数据可视化

---

## 功能模块

### 1. 📊 首页仪表盘
- 本月收支总览（收入/支出/结余）
- 净资产趋势图
- 当月预算消耗进度条
- 最近5笔交易记录
- **财务自由进度条**：被动收入 ÷ 月支出 = 自由百分比

### 2. 💰 记账（收支管理）
- 快速记一笔（收入/支出）
- 分类选择：餐饮、交通、购物、娱乐、住房、通讯、医疗、教育、人情、投资、工资、副业等
- 多账户支持：现金、银行卡、微信、支付宝
- 备注、日期、金额
- 支持编辑/删除
- **语音记账**（说一句"午饭花了20块"自动识别）

### 3. 📈 统计分析
- 月度收支对比图（柱状图）
- 支出分类饼图
- 月度趋势折线图
- 年度汇总
- 可筛选时间段

### 4. 🎯 预算管理
- 设置月度总预算
- 按分类设置子预算（吃饭1500、交通500等）
- 超支提醒
- 按月对比实际vs预算

### 5. 🏦 储蓄目标
- 创建储蓄目标（买房首付、买车、旅游、应急金）
- 设定目标金额和截止日期
- 进度条可视化
- 自动计算还需每月存多少

### 6. 🕊️ 财务自由规划
- 输入预期月支出
- 4%法则计算：需要本金 = 月支出 × 12 × 25
- 当前净资产距离自由目标差距
- 假设年化收益率，计算还需多少年
- "如果你每月存X元，年化Y%，Z年后实现财务自由"

### 7. 📅 账单提醒
- 信用卡还款日
- 房租/房贷
- 水电费
- 订阅服务（会员等）
- 到期前3天推送提醒

### 8. ⚙️ 设置
- 货币单位（人民币/美元）
- 数据备份/恢复（导出CSV/JSON）
- 分类自定义
- 密码锁
- 记账提醒（每天20:00弹一下）

## 技术架构

```
lib/
├── main.dart                    # 入口
├── app.dart                     # MaterialApp + 路由
├── core/
│   ├── database.dart            # SQLite 数据库初始化
│   ├── theme.dart               # 主题（深色+金色点缀）
│   └── constants.dart           # 常量（分类列表等）
├── models/
│   ├── transaction.dart         # 交易记录模型
│   ├── budget.dart              # 预算模型
│   ├── goal.dart                # 储蓄目标模型
│   ├── bill.dart                # 账单提醒模型
│   └── account.dart             # 账户模型
├── providers/
│   ├── transaction_provider.dart
│   ├── budget_provider.dart
│   ├── goal_provider.dart
│   └── settings_provider.dart
├── pages/
│   ├── home_page.dart           # 首页仪表盘
│   ├── add_transaction_page.dart # 记账
│   ├── statistics_page.dart     # 统计分析
│   ├── budget_page.dart         # 预算管理
│   ├── goals_page.dart          # 储蓄目标
│   ├── freedom_page.dart        # 财务自由
│   ├── bills_page.dart          # 账单提醒
│   └── settings_page.dart       # 设置
└── widgets/
    ├── summary_card.dart        # 汇总卡片
    ├── progress_bar.dart        # 进度条
    ├── transaction_tile.dart    # 交易条目
    └── chart_widgets.dart       # 图表组件
```
